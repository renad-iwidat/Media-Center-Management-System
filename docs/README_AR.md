# 📻 نظام إدارة مركز الإعلام

## 🎯 نظرة عامة

نظام متكامل لإدارة عمليات مركز الإعلام يتضمن:
- إدارة الموظفين والجداول الزمنية
- إدارة البرامج والحلقات
- إدارة الضيوف والمعلومات الشخصية
- تنظيم الأقسام والفرق
- إدارة السياسات التحريرية

## ✨ المميزات

### 👥 إدارة الموظفين
- إضافة وتعديل وحذف الموظفين
- تعيين أيام وساعات العمل
- تتبع معلومات الموظفين

### 📺 إدارة البرامج
- إنشاء وإدارة البرامج
- تعيين أوقات البث
- ربط الحلقات بالبرامج

### 🎬 إدارة الحلقات
- إضافة حلقات للبرامج
- تعيين أرقام وتواريخ الحلقات
- تتبع معلومات الحلقات

### 🎤 إدارة الضيوف
- إضافة معلومات الضيوف
- تخزين بيانات الاتصال
- البحث عن الضيوف

### 📋 تنظيم الهيكل
- إنشاء الأقسام (Desks)
- تنظيم الفرق داخل الأقسام
- تعيين المديرين

### 📜 السياسات التحريرية
- تعريف السياسات التحريرية
- تعيين القواعد والمبادئ
- تفعيل/تعطيل السياسات

## 🚀 البدء السريع

### المتطلبات
- Node.js (v18 أو أحدث)
- PostgreSQL
- متصفح ويب حديث

### التثبيت والتشغيل

**الخادم الخلفي (Terminal 1):**
```bash
npm install
npm run dev
```

**الواجهة الأمامية (Terminal 2):**
```bash
cd portal-frontend
npm install
npm run dev
```

**فتح البوابة:**
- اذهب إلى `http://localhost:3000`
