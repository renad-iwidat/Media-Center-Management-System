# ميزة عرض تاريخ آخر سحب للمصادر

## نظرة عامة
تم إضافة ميزة عرض تاريخ آخر سحب (Last Fetched) لكل مصدر في واجهة المصادر، مع تنسيق ذكي للوقت وإشارة بصرية للمصادر التي تم سحبها مؤخراً.

## التغييرات المطبقة

### 1. Backend Changes

#### `src/controllers/database/sources.controller.ts`
- تحسين دالة `getSourcesWithFetchInfo()` لتنسيق تاريخ آخر سحب بشكل نسبي:
  - **الآن**: إذا كان السحب قبل أقل من دقيقة
  - **منذ X دقيقة**: إذا كان السحب خلال الساعة الأخيرة
  - **منذ X ساعة**: إذا كان السحب خلال آخر 24 ساعة
  - **منذ X يوم**: إذا كان السحب خلال آخر 7 أيام
  - **تاريخ كامل**: للتواريخ الأقدم من أسبوع

- إضافة حقل `is_recently_fetched` للإشارة إلى المصادر التي تم سحبها خلال آخر ساعتين
- إضافة حقل `type` لعرض نوع المصدر

### 2. Frontend Changes

#### `frontend/src/services/api.ts`
- تغيير endpoint من `/data/sources` إلى `/data/sources/fetch-info/all` للحصول على معلومات آخر سحب

#### `frontend/src/components/news/SourcesView.tsx`
- إضافة tooltip عند hover على التاريخ لعرض التاريخ الكامل
- تحسين التنسيق البصري:
  - نقطة خضراء متحركة للمصادر التي تم سحبها مؤخراً
  - تغيير لون النص حسب حالة السحب (رمادي فاتح للمصادر المسحوبة، رمادي داكن للمصادر التي لم يتم سحبها)
  - نص مائل للمصادر التي لم يتم سحبها بعد

### 3. Database Integration

#### `src/services/database/database.service.ts`
- الدالة `updateLastFetched()` موجودة بالفعل وتقوم بتحديث `last_fetched_at` عند سحب الأخبار

#### `src/services/news/rss-pipeline.service.ts`
- يتم استدعاء `SourceService.updateLastFetched()` تلقائياً بعد كل عملية سحب ناجحة

## كيفية الاستخدام

1. افتح صفحة المصادر في الواجهة الأمامية
2. ستشاهد تاريخ آخر سحب لكل مصدر بتنسيق نسبي سهل القراءة
3. المصادر التي تم سحبها مؤخراً (خلال آخر ساعتين) ستظهر بنقطة خضراء متحركة
4. عند hover على التاريخ، سيظهر tooltip بالتاريخ الكامل
