# Media Center Portal Frontend

A modern React TypeScript frontend for managing media center operations.

## Features

- 👥 **Employee Management** - Add, edit, and delete employees with schedules
- 📺 **Program Management** - Create and manage programs with air times
- 🎬 **Episode Management** - Track episodes for each program
- 🎤 **Guest Management** - Manage guest information and contact details
- 📋 **Desk & Team Organization** - Organize teams within desks
- 📜 **Editorial Policies** - Define and manage editorial policies

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

The app will be available at `http://localhost:3000`

## Build

```bash
npm run build
```

## Configuration

Create a `.env` file based on `.env.example`:

```
VITE_API_URL=http://localhost:5000/api/portal
```

## Technologies

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Axios
- React Router
